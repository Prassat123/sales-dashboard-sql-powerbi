# Sales Dashboard Project

## Project Overview

This project analyzes sales transaction data to identify revenue trends, top-performing product categories, regional sales performance, and payment method usage.

## Tools Used

* SQL
* Excel
* GitHub

## Key Insights

* Electronics generated the highest sales revenue.
* Texas was the top-performing region.
* Credit Card and Debit Card were the most common payment methods.
